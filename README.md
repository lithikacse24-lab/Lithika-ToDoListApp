# 📝 To-Do List Web App

A simple, responsive, and user-friendly To-Do List web application built using HTML, CSS, and JavaScript. 
This project helps users organize and manage their daily tasks efficiently by providing features to add, complete, and delete tasks.

## Live Demo
https://lithikacse24-lab.github.io/DecodeLabs-Internship-Task1/

## 📌 Project Overview

The To-Do List Web App is designed to improve productivity by allowing users to keep track of their tasks in a clean and interactive interface. The application works seamlessly across desktop and mobile devices, ensuring a smooth user experience.

## 🚀 Features

- ➕ Add new tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete individual tasks
- 📱 Responsive design for mobile and desktop devices
- ⚡ Simple and intuitive user interface
- 🎯 Real-time task management

## 🛠️ Technologies Used

- **HTML5** – Provides the structure of the application
- **CSS3** – Handles styling, layout, and responsiveness
- **JavaScript** – Implements functionality and interactivity
