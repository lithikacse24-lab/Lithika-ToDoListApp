#📝 To-Do List Web App :

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. This project helps users manage daily tasks efficiently by allowing them to add, delete, and mark tasks as completed.

##🚀 Features
- ➕ Add new tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete individual tasks
- 📱 Fully responsive design (works on mobile and desktop)

##🛠️ Technologies Used
- HTML5 – Structure of the app
- CSS3 – Styling and layout
- JavaScript – Functionality and interactivity
